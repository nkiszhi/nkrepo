# 杀毒软件图标资源

本目录用于存放杀毒软件的图标文件。

## 图标命名规范

请按照以下命名规范添加图标文件:

- Avira: `avira.png`
- McAfee: `mcafee.png`
- WindowsDefender: `windows-defender.png`
- IkarusT3: `ikarus.png`
- Emsisoft: `emsisoft.png`
- FProtect: `f-prot.png`
- Vba32: `vba32.png`
- ClamAV: `clamav.png`
- Kaspersky: `kaspersky.png`
- ESET: `eset.png`
- DrWeb: `drweb.png`
- Avast: `avast.png`
- AVG: `avg.png`
- AdAware: `adaware.png`
- FSecure: `f-secure.png`

## 图标规格建议

- 格式: PNG 或 SVG
- 尺寸: 64x64 像素或更大
- 背景: 透明背景
- 风格: 官方logo

## 获取图标

可以从以下渠道获取杀毒软件官方logo:

1. 各杀毒软件官方网站
2. https://www.virustotal.com/gui/ (查看检测报告中的引擎图标)
3. https://commons.wikimedia.org/
4. 使用搜索引擎搜索 "杀毒软件名称 logo png"

## 注意事项

- 如果没有找到某个杀软的图标,系统会自动使用 `default.svg` 作为默认图标
- 图标文件名必须与上述命名规范完全一致(包括大小写)
- 建议使用PNG格式以保证最佳兼容性
